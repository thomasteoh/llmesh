Run

macOS: remove quarantine before first run

xattr -cr llmesh-client
Config file

Create a config.yaml alongside the binary. A pre-filled file with your token is available above after generating a token.

router_url: "wss://{{.Host}}/ws/client" router_token: "ct-yourname-yourtoken" # from Clients page max_concurrent: 4 # parallel jobs this machine handles models: - name: "llama3.2:3b" endpoint: "http://localhost:8080" # llama-server base URL - name: "unsloth/qwen3-30b-a3b" endpoint: "http://localhost:8081" chat_template: "qwen2.5" # optional template override # Aliases are managed server-side in the admin UI — Clients page.

Config file

Create a shim-config.yaml alongside the binary. A pre-filled file with your token is available above after generating a token. API keys are read from environment variables.

router_url: "wss://{{.Host}}/ws/client" router_token: "ct-yourname-yourtoken" max_concurrent: 4 models: - name: "gpt-4o" context_size: 128000 backend: type: http url: "https://api.openai.com" format: openai auth_type: bearer auth_value: "${OPENAI_API_KEY}" - name: "claude-sonnet-4-5" context_size: 200000 backend: type: http url: "https://api.anthropic.com" format: anthropic auth_type: bearer auth_value: "${ANTHROPIC_API_KEY}" # - name: "my-model" # backend: # type: command # command: "/path/to/adapter.sh"
